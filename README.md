# Prakriti.AI

Prakriti.AI is a municipal waste operations and carbon intelligence platform designed for city governments, waste workers, bulk waste generators, processors, and auditors.

## Core goals

- Waste operations management
- Route and field execution tracking
- Source traceability
- Bulk generator compliance
- Processing and recovery logging
- Carbon accounting and audit trails
- Ward and city analytics

## Project structure

- `backend` - FastAPI backend
- `frontend` - Next.js web application
- `docs` - product and technical documentation
- `infrastructure` - deployment and container setup
